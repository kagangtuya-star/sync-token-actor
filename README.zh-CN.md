# Sync Token Actor（简体中文）

同步角色与标记名称/图片，并在角色卡片头像区域提供简洁的可交互面板。

## 功能列表
- 角色名称变化后自动写入原型标记，并可级联到场景中的相关标记。
- 可选的图片自动同步，保持标记图像与角色头像一致。
- 可在同步时关闭角色环，避免遮挡底部绘制（特别适合使用大量美术资源的预制包）。
- 角色卡片头像处显示标记缩略图 + 同步按钮 + “始终同步”链条按钮。
- 在拖入标记前即写入最新名称和图片，确保场景保持干净一致。
- 内置中英本地化，所有提示、设置说明均有中文版本。

## 配置步骤
1. 打开 **设置 → 模块设置 → Sync Token Actor**。
2. 根据需求开启或关闭名称/图片自动同步。
3. 若需对场景中已存在的标记生效，可勾选“同步场景标记”。
4. 客户端可独立设置同步面板是否显示以及在头像上的显示位置。
5. 如需在同步时关闭角色环，只需勾选对应设置即可。
6. 需要某个角色始终保持同步时，可在头像叠层上点击链条图标，强制此角色启用自动同步。

## UI 面板
- 鼠标悬浮角色头像即可显示面板。
- 点击标记缩略图可查看原图并直接打开原型 **Token Configuration**。
- 同步按钮可即时用角色头像覆盖标记图片。
- 链接按钮用于切换“始终同步”，开启后即使全局自动同步关闭，该角色也会持续同步。
